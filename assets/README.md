# CryptoWatch Assets

## 项目资源文件

### 可添加的资源

- `logo.png` - 项目 Logo（建议 512x512）
- `banner.png` - GitHub 横幅（建议 1280x640）
- `screenshot.png` - 功能截图
- `architecture.png` - 架构图

### 使用方式

在 README.md 中引用：
```markdown
![CryptoWatch Banner](assets/banner.png)
```

### 设计建议

- 使用蓝色/紫色主题（加密货币风格）
- 包含关键元素：📊 图表、🤖 AI、⛓️ 区块链
- 保持简洁专业
